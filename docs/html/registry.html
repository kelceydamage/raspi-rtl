Registry¶

Thhis module provides dynamically loads tasks as provided in the raspi-tasks package.

Note

This is not intented to be used for any other reason. Standard python standard python importing should always be used.

rtl.transport.registry._loader(path)¶

Load all modules located within the given path. Add each loaded module to a dict of modules.

Warning

This is a private function and should never be called directly.

Parameters

path (str) – path to a folder containing modules.

Returns

a map of names and module references.

Return type

dict

rtl.transport.registry.import_tasks(module_name)¶

Convert module name into a path and call loader on the path.

Example

modules = import_tasks('rtl.tasks.*')
Parameters

module_name (str) – either a path or a name referencing a valid module.

Returns

a map of names and module references.

Return type

dict
